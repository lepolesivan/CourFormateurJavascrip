Une fonction permet d'enregistrer des instructions qui seront exécutées quand on en aura besoin. Une fonction peut servir à faire des calculs (comme les fonctions mathématiques). Dans ce cas, la fonction retourne une valeur. Quand la fonction sera appelée, elle pourra être utilisée comme une expression, puisqu'elle retourne une valeur.

Déclaration d'une fonction

Syntaxe :

        
            function nomDeFonction(argument1,argument2,...){
                ...
                return expression
            }

Appeler une fonction

Syntaxe :

        
            nomDeFonction(valeur1,valeur2,...);
            let variable = nomDeFonction(valeur1,valeur2,...);

Les variable déclarées dans les fonctions n'existent que dans ces fonction.
